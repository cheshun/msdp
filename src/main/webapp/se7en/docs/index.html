图标

Se7en 自定义图标

Se7en的自定义图标通过fonticons实现，通过预定的规则可用于任何页面上。如： 

<span aria-hidden='true' class='se7en-icon-name'></span>

以下是se7en自定义图标中包含的所有图标:



这些图标可以被无限的放大且不失真，并可以通过css改为任何颜色。

Font Awesome Icons

我们已经包含了最新的字体和最好的图标，其中包括超过360个以上的图标。要使用这些图标只需要添加一个class即可： 

<span class='icon-'></span>

例如： 

<span class='icon-home'></span>

需要了解更多信息，请访问：Font Awesome.

Glyphicons

我们还包含了180个Glyphicons和Glyphicon Halflings图标，需要注意的是Glyphicon Halflings通常是不免费提供的。

如果需要使用Font Awesome icon，添加以下class即可： 

<span class='glyphicon glyphicon-'></span>

例如： 

<span class='glyphicon glyphicon-user'></span>

如果需要了解更多如何使用Glyphicons图标的信息，请访问Bootstrap's Documentation.
